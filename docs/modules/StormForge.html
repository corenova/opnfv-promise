StormForge Module

Defined in: src/stormforge.coffee:9

Provides the VIM Extensibe services

This module provides the following classes:

  • ComputeCapacity
  • ComputeService
  • Hypervisor
  • ProviderService
  • ResourceElement
  • ResourcePool
  • ResourceProvider
  • ResourceReservation
  • ServiceCapacity
  • StormForge
  • StormForgeAsset
  • StormForgeResource

This module is a rollup of the following modules:

  • ResourceProvider
    Capacity of the compute in cores, ram and instances
  • ResourceReservation
    Resource reservation data model
  • StormForgeAsset
    StormForge Asset describes the actual instance properties such as the status, IP Address to reach etc.,
  • StormForgeResource
    A pool of resources from various Resource Providers
